## NVM 
1. NVM [다운로드](https://github.com/creationix/nvm)
2. NVM을 사용하여 node.js 설치  
   * 설치할 node [버전](https://nodejs.org/en/) 확인  
   * NVM으로 node 설치 후 nvm use 명령을 통해 node 활성화  
3. node-sass 설치 및 버전 확인
   * npm install --global node-sass 
   * node-sass -v
